个人博客系统后端 (Go + Gin + GORM)
一个基于 Go 语言开发的个人博客后端 API 服务，支持用户认证、文章管理和评论功能，采用 Gin 框架处理 HTTP 请求，GORM 库进行数据库操作，JWT 实现身份认证。
功能特点
用户系统：注册、登录功能，密码加密存储，JWT 身份验证
文章管理：支持博客文章的创建、查询、更新、删除（CRUD）
评论功能：已登录用户可对文章发表评论，支持查询文章评论列表
权限控制：仅文章作者可修改或删除自己的文章
错误处理：统一错误响应格式，包含合适的 HTTP 状态码
日志记录：使用 logrus 记录系统运行日志和错误信息
技术栈
编程语言：Go 1.19+
Web 框架：Gin v1.9+
ORM 库：GORM v2.0+
数据库：MySQL 8.0+
身份认证：JWT (golang-jwt/jwt v5)
日志：logrus
配置管理：godotenv
